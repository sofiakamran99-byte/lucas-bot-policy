Privacy Policy for Lucas Bot

Last updated: November 2025

Thank you for using Lucas Bot! This Privacy Policy explains how Lucas Bot handles user information within Discord.

1. Data Collection

Lucas Bot does not collect, store, or share any personal data from users. All data processed by the bot is limited to what is required for its commands and features to function properly within Discord servers.

2. Temporary Data

Some temporary information (like message content, user IDs, or server IDs) may be used only in memory during command execution. This data is never saved, logged, or shared with third parties.

3. Third-Party Services

Lucas Bot does not use any third-party analytics or external APIs that collect user information.

4. User Rights

If you have any concerns, questions, or requests related to data, you can contact the developer using the information below.
